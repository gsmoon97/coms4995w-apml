## Setup & Reproducing Results

### 1. Dataset

The notebook expects the Kaggle dataset in a `home-credit-default-risk/` folder at the same level as `a2.ipynb`. To set this up:

1. Download the dataset from <https://www.kaggle.com/c/home-credit-default-risk/data>.
2. Unzip the downloaded archive.
3. Place the CSV files directly under `assignments/a2/home-credit-default-risk/`.

The primary file used is `application_train.csv` (the main application table).

### 2. Dependencies

The notebook was developed with **Python 3.10** and requires:

- `numpy`
- `pandas`
- `matplotlib`
- `seaborn`
- `scikit-learn`
- `xgboost`

Install via:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn xgboost
```

### 3. Running the Notebook

Open `a2.ipynb` and run all cells sequentially. The notebook will:

1. Load and preprocess the dataset (anomaly correction, missing-value treatment, feature engineering, encoding, train/val/test split).
2. Train and evaluate GBDT (XGBoost) with learning-rate sweeps, `n_estimators` sensitivity, and `RandomizedSearchCV` tuning.
3. Train and evaluate MLP (scikit-learn `MLPClassifier`) across architectures, activations, learning rates, and regularization strengths.
4. Produce a side-by-side comparison (metrics table, PR/ROC curves, threshold analysis).
5. Save all figures to `figures/` and metrics to `results.json`.
